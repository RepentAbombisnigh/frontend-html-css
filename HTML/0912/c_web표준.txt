# 웹 표준의 정의
: 웹에서 사용되는 표준화된 기술이나 규칙을 의미

# 웹 표준의 중요성
: 다양한 기기와 브라우저엣거 일관된 웹 경험을 제공
: 웹 접급성 보장
: 유지 보수 용이

# 웹 접근성
: 모든 사용자가 웹사이트와 도구에 동등하게 접근할 수 있게 만드는 것
: 장애인, 고령자를 포함한 모든 사용자의 웹 이용 확장

# 웹 표준의 주요 기술
: HTML : 웹 페이지의 기본 구조와 내용을 정의 - HTML5
: CSS : 웹 페이지의 디자인 레이아웃 애니메이션 정의 - CSS3
: Javascript : 웹 페이지의 동적인 요소를 추가
: DOM : 웹 페이지의 객체와 구조를 표현하고 JS(Javascript)가 페이지와 상호작용하도록 도와주는 인터페이스
: Accessibillity guidelines : 웹 컨텐츠를 모든 사용들이 접근하고 이해할 수 있도록 하는 웹 접근성 가이드라인

# Interface
: 상호작용(할 수 있도록 미리 규정한 내용)
: EXP, # 뒤에 내용을 입력한다. 라고 적용하고 위와 같이 내용 입력
